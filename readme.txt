Блок питания БП1 для аппаратуры АВАНТ.
Версия платы: 03v2.
Дата создания: октябрь 2014.
Среда разработки: OrCAD 16.6-S036.
База: greysql.

Измеренные пороги:
Включение - 158 В.
Выключение - 132 В. 

Надо:
1. Сделать сборку на блок. Длина изолирующей трубки 18см + хомутик.

Проблемы:
1. Контактные площадки TP9-TP14 имеют слишкой узкий ободок, который на производстве увеличили до 1.5мм.
2. При испытаниях на плавное снижение/нарастание питания, при процедуре (220В, 220В->0В, 0В, 0В->220В каждый интервал по 1мин) наблюдалось кратковременное включение при нарастании в районе 60В. Если проводилось испытание только плавного нарастания (0->220В 2мин, 0->80В 2мин) такого не наблюдалось. Для "уменьшения" проблемы, были изменены номиналы конденсаторов С7 и С9 до 0,01мкФ.

Изменения:
1. Перенесена часть конденсаторов с 48В на 220В. 
2. По питанию 220В сделаны пороги включения (порядка 150 В) и отключения (порядка 120 В).
3. Выводные элементы заменены на планарные.
